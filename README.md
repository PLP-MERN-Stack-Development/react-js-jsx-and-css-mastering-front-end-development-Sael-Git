# React + Tailwind Dashboard (Assignment-ready)

This is a small professional-looking dashboard project built with:
- React (Vite)
- Tailwind CSS
- Axios for API calls
- Recharts for simple charts

## Features
- Component architecture: `components/`, `pages/`, `sections/`, `hooks/`, `api/`
- State management using React hooks (`useState`, `useEffect`)
- Integrates with external APIs:
  - JSONPlaceholder for users and posts
  - CoinGecko for current crypto prices
- Responsive layout using Tailwind CSS
- Ready for deployment (Vercel / Netlify)

## Quick start (local)
1. Clone or download the project folder.
2. Install dependencies:
   ```
   npm install
   ```
3. Initialize Tailwind if needed (optional if you keep provided config):
   ```
   npx tailwindcss init -p
   ```
4. Start dev server:
   ```
   npm run dev
   ```
5. Build:
   ```
   npm run build
   ```
